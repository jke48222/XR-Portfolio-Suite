# com.jalen.xrshared

Shared core package for the XR Portfolio Suite.

Contents:
- AppBootstrap and AppConfig
- ServiceLocator and deterministic startup
- TickRunner for centralized updates
- DebugOverlay and DebugStats
- PerformanceGovernor and CaptureController
- Persistence store with schema versioning
- Minimal audio features scaffold

This package is designed to be extended per app with app-specific implementations.
